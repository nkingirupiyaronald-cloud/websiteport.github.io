SQL · Data Engineering
Nashville Housing — Data Cleaning in SQL

Cleaned a Nashville real-estate dataset in SQL Server across five stages: standardising sale date formats, backfilling null property addresses via a self-join on ParcelID, splitting combined address strings into separate city/state columns using SUBSTRING and PARSENAME, standardising boolean flags (Y/N → Yes/No), and removing duplicate rows with a ROW_NUMBER() CTE before dropping redundant columns.

SQL · Exploratory Analysis
Covid-19 Global Data Exploration

Explored a global Covid-19 dataset across deaths and vaccinations tables, calculating death rates per case, infection rates as a share of population, and a rolling vaccination count using SUM() OVER(PARTITION BY). Demonstrated CTEs, temp tables, joins across two datasets, and created a PercentPopulationVaccinated view for downstream visualisation.
